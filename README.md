# chip8emu

A C99 implementation of CHIP8 emulation library

## Requirements

* C/C++ compilier
* CMake

## Quick started

```
git clone https://github.com/thaolt/chip8emulator.git
cd chip8emulator
git submodule update --init
mkdir build && cd build
cmake ..
make
cd emu-termbox
./emu-termbox
```
